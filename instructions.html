Game Instructions

1. Press any key to start the game.

2. The computer will create a colour pattern that you need to repeat everytime. For each level the computer will add one colour to the pattern.

3. Play the pattern by pressing the colour button in correct order.

4. Once you complate one level the computer will add one colour to the existing pattern which you need to repeat.

5. The game ends if you press a wrong colour.
